# SquashWorld Website

This is a simple static website project created using **HTML and CSS**.

The project was developed for educational purposes and demonstrates basic web development concepts such as page structure, navigation menus, tables, images, and styling with CSS.

## Languages Used
- HTML  
- CSS  

## Project Description
The website presents information about the sport of **squash**, including:
- Rules of the game
- Equipment and products
- An e-shop style product list
- A featured product page
- Contact information

The focus of the project is on practicing **HTML structure and CSS styling**, without using JavaScript or backend technologies.

## Purpose
- Practice basic HTML & CSS
- Learn page layout and navigation
- Create a simple multi-page website

## How to Use
Simply open any `.html` file in a web browser.
